◆ checkHydroHotStart()

bool Antares::Data::Study::checkHydroHotStart ( )

In case hydro hot start is enabled, checking all conditions are met.

If hydro hot start is enabled, check that :

  • For all areas for which reservoir management is enabled :
    • Their starting level is initialized on the same day
    • This day is the first day of the simulation calendar
  • The simulation lasts exactly one year
  • All batches (or sets) of simultaneous years have the same size (obvious if a parallel run is not required : answer is yes).

If these conditions are not met, some error message is raised, when attempting to run the study.

◆ importTimeseriesIntoInput()

bool Antares::Data::Study::importTimeseriesIntoInput ( )

Import all time-series into the input folder.

This method should be used when the solver has finished its simulation and the data generated by the preprocessors must be imported in the input folder. The selection of time-series is controlled by the general data (timeSeriesToImport). When calling this routine, and if timeSeriesToImport is not equals to 0, the input folder will be modified.
