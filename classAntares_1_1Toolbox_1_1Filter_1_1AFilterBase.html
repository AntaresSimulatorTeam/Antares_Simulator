Detailed Description

Abstract Filter.

A filter is a single test/condition to know if a value should be displayed and/or managed according the user-settings (for example hour in the year > 42 is a filter). A filter is mainly used by datagrid.

A filter is composed into three parts : Its name/caption, its operator (equals, less than...) and the parameters (a single one most of the time). Graphically, it will be the same. All components created by the filter only handle data specific to the filter itself.
