Detailed Description

A list of areas.

Here is one way to iterate through the list :

AreaList* l;
...
<do some stuff here to fill the area list>
...
Area* c = l->first;
while (NULL != c)
{
...
<do some stuff here with the area via `c`>
...
c = c->next;
}
