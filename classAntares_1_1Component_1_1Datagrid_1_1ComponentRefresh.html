void Antares::Component::Datagrid::ComponentRefresh::forceRefresh ( )

Force a complete refresh of the grid.

The sub-component Grid (wxGrid) caches some values, like the size of the grid, and the name of the columns. This method is especially useful when any important changes occur (for example another area has just been selected) and the entire grid is invalidated.
