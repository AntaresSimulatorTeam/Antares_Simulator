◆ removeDisabledClusters()

uint Antares::Data::PartRenewable::removeDisabledClusters ( )

Removes disabled renewable clusters.

All clusters with the flag 'enabled' turned to false will be removed from 'list'. As a consequence, they will no longer be seen as renewable clusters from the solver's point of view.

Warning
This method should only be used from the solver
Returns
The number of disabled clusters found

std::vector<RenewableCluster*> Antares::Data::PartRenewable::clusters

All clusters for the area.

This variable is only available at runtime from the solver. It is initialized in the same time that the runtime data.

This list is mainly used to ensure the same order of the renewable clusters in the outputs.
