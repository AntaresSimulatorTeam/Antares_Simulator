Create and initialize the list of all clusters (with the mustrun flag or not)

This method must be called before prepareClustersInMustRunMode() to ensure the same order whatever the value of the 'mustrun' flag is.

◆ prepareClustersInMustRunMode()

uint Antares::Data::PartThermal::prepareClustersInMustRunMode ( )

Prepare all thermal clusters in the 'mustrun' mode.

All clusters with the flag will be moved to the container 'mustrunList'. As a consequence, they will no longer be seen as thermal clusters from the solver's point of view.

Warning
This method should only be used from the solver
Returns
The number of clusters found in 'must-run' mode

◆ removeDisabledClusters()

uint Antares::Data::PartThermal::removeDisabledClusters ( )

Removes disabled thermal clusters.

All clusters with the flag 'enabled' turned to false will be removed from 'list'. As a consequence, they will no longer be seen as thermal clusters from the solver's point of view.

Warning
This method should only be used from the solver
Returns
The number of disabled clusters found

std::vector<ThermalCluster*> Antares::Data::PartThermal::clusters

All clusters for the area.

This variable is only available at runtime from the solver. It is initialized in the same time that the runtime data.

This list is mainly used to ensure the same order of the thermal clusters in the outputs.
