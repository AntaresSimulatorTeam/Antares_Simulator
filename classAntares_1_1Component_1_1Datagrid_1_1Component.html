Detailed Description

A datagrid with virtual values.

This datagrid only provides a common interface to deal with 1D-arrays and matrices used by the solver. It does not store any values, and it is controlled by a renderer, which is the only one able to provide informations about the grid (values, column names...).

In the most cases, the renderer is attached to an input selector (for example an area) to update the GUI accordinly.
