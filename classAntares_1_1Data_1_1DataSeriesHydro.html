◆ count

uint Antares::Data::DataSeriesHydro::count

The number of time-series.

This value must be the same as the width of the matrices mod and fatal. It is only provided for convenience to avoid same strange and ambiguous code (for example using fatal.width and mod.width in the same routine, it might indicate that the two values are not strictly equal)
